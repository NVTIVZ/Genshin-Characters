
# Genshin Characters




## Tech Stack

**Client:** React, ChakraUI, React-query




## Run Locally

Clone the project

```bash
  git clone https://github.com/NVTIVZ/Genshin-Characters
```

Go to the Front-end directory

```bash
  cd genshin-characters
```

Install dependencies

```bash
  npm install
```

Start the Front-end

```bash
  npm run start
```
